Terms: Maturity of Organization

Level 1

AI-Driven

Amplify existing workflows. Doing the same things, but faster using AI tools.

Ex: Copilot for code, ChatGPT for emails.
Level 2

AI-First

Redesign operations. Processes are built assuming AI is primary actor, human is reviewer.

Ex: Automated PR review, AI-gen SPEC to Tests.
Level 3

AI-Native

AI is the Product. The core value proposition wouldn't exist without generative AI models.

Ex: Midjourney, Perplexity, Auto Agents.

Module 8: Economics of an AI Product

Margin = Price ‚àí Variable Cost (COGS)

Usage (Pay-as-you-go)

Users pay per token/generation.

Best when: Costs are unpredictable & high per action.

Subscription

Fixed MRR, limits applied.

Best when: Usage averages out, retention is key.

Enterprise (SaaS)

High ticket, custom SLAs.

Best when: Value is workflow integration + security.
